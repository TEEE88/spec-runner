# 状態遷移図

```yaml
エンティティ:
  - name: "{エンティティ名}"
    状態:
      - 状態: "{状態名}"
        説明: "{状態の意味}"
        操作: "{許可される操作}"
    遷移:
      - from: "{初期状態}"
        to: "{状態A}"
        when: "{イベント}"
      - from: "{状態A}"
        to: "{状態B}"
        when: "{イベント}"
```
